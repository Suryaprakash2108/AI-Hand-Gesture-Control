🖐️ AI Hand Gesture Computer Control – Computer Vision Project

This project uses Computer Vision and Hand Tracking techniques to control computer operations using real-time hand gestures. Leveraging OpenCV, MediaPipe, and PyAutoGUI, the system enables users to interact with their computer without physical contact through webcam-based gesture recognition.

🔬 Built as part of an AI & Computer Vision project by **Mopada Surya Prakash**, UG Scholar at Mohan Babu University, Tirupati, India.

---

# 🧭 Project Overview

### Problem Statement:

Traditional computer interaction methods such as mouse and keyboard require physical contact. In touchless environments and smart interaction systems, gesture-based control offers a more intuitive and futuristic solution.

### Solution:

This project uses real-time webcam input and MediaPipe hand tracking to detect hand landmarks and perform system operations based on finger gestures.

### Use Case:

Useful for:

* Touchless computer interaction
* Smart automation systems
* AI-based human-computer interaction
* Accessibility applications
* Gesture-controlled environments

# 🧱 System Architecture

### Workflow:

1. Webcam captures live video feed
2. OpenCV processes video frames
3. MediaPipe detects hand landmarks
4. Finger positions are analyzed
5. Gestures are mapped to computer actions
6. PyAutoGUI performs system control operations
